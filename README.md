# Simple Calculator

A basic calculator built using **HTML** and **CSS**. This project demonstrates a simple calculator interface with a clean and responsive design.

## Features

* Clean and modern user interface
* Responsive layout
* Calculator buttons for numbers and basic operations
* Beginner-friendly project

## Technologies Used

* HTML5
* CSS3

## Project Structure

```text
calculator/
├── index.html
└── style.css
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
